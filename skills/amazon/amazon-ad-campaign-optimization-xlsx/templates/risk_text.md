# Sheet 08 风险提示文案模板

> 此文件由 `build_xlsx.py` 直接读取渲染，不进 LLM 输出 schema。
> 占位符 `{{...}}` 由 build_xlsx 在写入前根据 plan.meta 替换。

## 第一段：绝对不能做的事（红底 #FFCCCC）

| 风险项 | 为什么不能做 | 正确做法 |
|---|---|---|
| 否定核心词同义词（trail cameras/trail cam 等同根变体） | 切断 A9 相关性评分，自然位会掉 | 同义词全保，只否明确无关词 |
| 一次性把 bid 改到 < $2（或当前 bid 的 -50% 以上） | 会立刻掉出首页，回来需 2-3 周 | 分步调，一次 ±20% 内 |
| 改完广告 5 天内频繁调整 | A9 需要时间学习，频调数据归零 | 调整后至少稳定 5-7 天观察 |
| 把核心词 Campaign 删掉重建 | 历史数据清零，从零学习 | 只能修改，不能删除 |
| 为 ACOS 关掉核心词广告 | 广告一停 3-5 天自然位开始掉 | 冲量期看 CTR 和单量增速 |
| 盲目新增无关关键词 | 稀释 Campaign 相关性评分 | 只加已验证转化词 + 战略保护词 |
| 投放 SBV 视频广告或参加 Vine | 用户既定策略不投 | 走 SP 精准/词组/广泛 + 自然位 + Coupon |

## 第二段：出现这些信号立即调整（金底 #FFE699）

| 警告信号 | 可能原因 | 应对方案 |
|---|---|---|
| {{核心词}} 首页广告平均位置倒退 | 顶部溢价不够 or 对手加价 | 顶部溢价 +150% → +200% |
| {{tos_campaign_name_or_default}} 日预算提前烧完 | 溢价高 + 顶部曝光好（好事） | 观察 3 天后加预算 |
| CTR 下降（< 0.5%） | 主图/价格被竞品压制 | 立即优化主图（CTR 杠杆优先级最高） |
| 自动广告新词持续没转化 | 拓词能力下降 | 降自动预算，转给词组 |
| 词组 / 广泛 ACOS 突然翻倍 | 烧出无关长尾词 | 立即导报告，补否定 |
| Listing 转化率（CVR）突然 < 类目均值 50% | A+/视频被竞品反超，或差评涌入 | 先查近 7 天差评，再判 Listing 内容 |

## 占位符默认值

| 占位符 | 含义 | 缺省回退 |
|---|---|---|
| `{{核心词}}` | 用户的核心大词，从 protected_keywords[0] 或 keyword_blackholes[0].keyword 取 | "核心大词" |
| `{{tos_campaign_name_or_default}}` | 新建 ToS-Only campaign 名，从 new_campaigns[0].name 取（含 "ToS"/"Top" 字样的） | "新建顶部专属 Campaign" |
